# IP-01 — Текущее исследование

Рабочая папка исследования **IP-01**.

Исследование проводится поэтапно. Каждый документ фиксирует отдельный шаг работы: постановку вопроса, промежуточные наблюдения, выводы исследовательской комиссии, редакционные решения и последующие уточнения.

## Логика нумерации

Формат имени файла:

```text
doc-001_0100.md
```

где:

* **doc** — рабочий документ исследования;
* **001** — номер исследовательского блока;
* **0100**, **0200**, **0300**... — последовательные этапы внутри исследования.

Нумерация ведётся с добавлением ведущих нулей, чтобы GitHub автоматически сортировал документы в правильном порядке.

---

## Принцип работы

Рабочий процесс построен следующим образом:

1. Автор подготавливает очередной документ.
2. Документ сохраняется в репозитории.
3. После публикации ссылки считается, что авторская часть этапа завершена.
4. Исследовательская работа продолжается со следующего этапа без повторного описания процесса.

---

## Статусы документов

| Статус       | Назначение                                |
| ------------ | ----------------------------------------- |
| **active**   | Документ участвует в текущем исследовании |
| **review**   | Документ находится на проверке            |
| **accepted** | Документ принят как рабочий результат     |
| **archive**  | Документ выведен из активной работы       |

## Назначение папки

Данная директория предназначена для хранения материалов текущего исследования.

В ней фиксируются:

* постановка исследовательских вопросов;
* промежуточные результаты;
* замечания исследовательской комиссии;
* редакционные решения;
* итоговые выводы.

Материалы являются рабочими документами и служат основой для подготовки будущих публикаций, исследований и книг проекта **In Profundo**.
