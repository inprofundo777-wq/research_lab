# IP-01 Pilot
## Первичная проверка исследовательской карточки

## 1. Цель пилота

Проверить не гипотезу целиком, а работоспособность инструмента исследования.

## 2. Корпус пилота

- Деян. 2:47
- 1 Кор. 3:5–9
- Гал. 2:20

## 3. Исследовательский вопрос

Какими литературными и богословскими средствами авторы Нового Завета переориентируют понимание событий от их видимого хода к действию Бога, Христа или Святого Духа?

## 4. Рабочая гипотеза

В некоторых текстах Нового Завета авторы сознательно переводят внимание читателя от видимого человеческого действия к богословскому субъекту происходящего.

## 5. Исследовательская карточка

| Поле | Описание |
|---|---|
| Текст | Исследуемый отрывок |
| Событие | Что происходит |
| Видимый субъект | Кто внешне действует |
| Богословский субъект | Кому автор приписывает действие |
| Тип атрибуции | Явная / косвенная / нарративная / экзистенциальная |
| Тип переосмысления | Субъект / причинность / центр внимания / результат |
| Что изменится без атрибуции | Как текст читался бы без авторской интерпретации |
| Предварительный вывод | Только локальный вывод по тексту |
